# Monasti
Sleek and Ultra performant BetterDiscord Theme


Monasti is a simple theme that uses a material-ish design and reuses the Discord background color variables, not the colors:
`--background-primary`
`--background-secondary`
`--background-tertiary`
`--background-floating`
